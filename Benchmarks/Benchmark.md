
# Benchmarks

This folder contain all benchmarks created for the purpose of this project. 
The goal is to benchmark all performance critical part of the software and
measure if after each release performance is decreased of increased




### Runing Benchmark

To save result of running benchmark run compiled application that can be found in
```out/build/-preset-name-/Benchmarks``` with options listed bellow
```--benchmark_out_format=json --benchmark_out=benchmark_example.json```
